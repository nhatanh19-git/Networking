# OSI Layers

## Giới thiệu

Mô hình OSI gồm 7 tầng:
1. Physical
2. Data Link
3. Network
4. Transport
5. Session
6. Presentation
7. Application

## Vai trò

- Layer 1: Tầng vật lý — truyền tín hiệu điện.
- Layer 2: Tầng liên kết dữ liệu — địa chỉ MAC, VLAN.
- Layer 3: Tầng mạng — IP, định tuyến.
...

## Ghi chú

- Switch hoạt động Layer 2.
- Router hoạt động Layer 3.
